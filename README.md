Mortgage Repayment Calculator

A simple **Mortgage Calculator web application** built using **HTML, CSS, and JavaScript**. It allows users to calculate their **monthly mortgage repayment** and the **total repayment amount** based on loan details.

Features

* Enter mortgage amount, term, and interest rate
* Choose mortgage type (Repayment or Interest Only)
* Calculates monthly and total repayment
* Input validation with error messages
* Responsive design

Project Files

* `index.html` – Structure of the calculator 
* `style.css` – Styling and layout 
* `main.js` – Calculation logic and validation
